# Analytics Vendors Updates Tracker

A lightweight, automated daily tracker for monitoring product releases from major analytics vendors, including Google Analytics, Adobe CJA, and Amplitude.

## Stack

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js with Axios and Cheerio for web scraping
- **Deployment**: GitHub Pages, GitHub Actions
